Seriously, what IS this?

A programming language that compiles to CPython bytecode, much like Scala compiles to JVM's. That essentially means that dg is an alternative syntax for Python 3. It allows you to use all of the existing libraries, too.

Check out all these awesome features

import '/asyncio'

main = async $ loop ->
  task = "Hello, {}!".format whatever where
    await asyncio.sleep 1
    whatever = "World"

  await task

loop = asyncio.get_event_loop!
loop.run_until_complete $ main loop

f = x -> raise if
  x :: int  => IndexError x
  otherwise => ValueError x

g = x -> except
  grr => f x
  grr :: IndexError => 0

except
  grr => g 0
  grr :: ValueError => 1

Proper error handling!

Remember that time you tried to write something in C? How your code was a mess of if-s that check whether the last call actually succeeded? No more! With an exception™ system, you no longer have to do that by hand. Only handle what you can, and let the code above do the rest!
